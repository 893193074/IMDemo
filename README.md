
项目说明
------

* 一个简单的即时通信客户端，由MySQL数据库 + Openfire服务器 + Android客户端 ( Smack 4.19 ) 组成
* 主要功能有注册账号、登录和退出系统、获取好友列表、添加和删除好友、收发文本消息、分享位置（基于百度地图SDK）
* 如果有疑问，可以发邮件联系我，如果觉得还不错，请点个 Star 支持一下，O(∩_∩)O...
